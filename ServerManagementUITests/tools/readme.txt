Steps to run SWTBot test case for GSServerManagement Plugin

1. copy the PSL eclipse plugin and the GigaSpaces plugin to <eclipse-home>/plugins directory
1. Install SWTBot from eclipse using update URL : http://download.eclipse.org/technology/swtbot/helios/dev-build/update-site
2. Install Ant
3. Make sure ANT_HOME variable is set to ant root folder
5. run run.bat while passing him your eclipse's root directory as the only argument
